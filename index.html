Listen

Es gibt verschiedene Arten von Listen

Geordnete Liste

Eine geordnete Liste ist eine bloße Nummerierung
Sie wird mit -ol- angegeben. Für jeden Listenpunkt muss -li- verwendet werden

  1. Informatik
  2. Geschichte
  3. Geographie
  4. Englisch
  5. Latein
  6. Zeichnen

Ungeordnete Liste

Eine ungeordnete Liste ist eine Aufzählung. (z.B. mit Punkten, Quadraten, etc.)
Diese Art wird gerne für Menüs verwendet (ohne Aufzählungspunkte)
Dazu braucht man die Tags -ul- und -li-.

Links

Was wäre eien Website ohne Links auf Unterseiten oder zu anderen Websites?
Auch inner halb einer Seite können wir einen Link (Anker) setzen. So können wir z.B. mit einem Klick wieder ganz nach oben zu gelangen.
Aufgebaut ist jeder Link mit dem Tag -a-

Attribute von Tags

Attribute bestimmen Tags näher. Um einen Link einzufügen is nicht nur der Tag -a- notwendig, sodern wir müssen ein Attribut (href) mit dem Link dazu angeben.
Für verschiedene Tags gibt es verschiedene Attribute.
